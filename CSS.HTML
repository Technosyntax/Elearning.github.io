1. Introduction to CSS

CSS stands for Cascading Style Sheets. It is a stylesheet language used for describing the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

While HTML provides the structure and content, CSS is responsible for the look and feel of your website â€“ colors, fonts, layouts, animations, and more!

2. Basic CSS Syntax

A CSS rule-set consists of a selector and a declaration block. 

/* This is a CSS comment */
selector {
    property: value; /* Declaration */
    property-two: value-two;
}
  • Selector: Points to the HTML element you want to style (e.g., h1, p, .my-class, #my-id).
  • Declaration: Consists of a property and a value, separated by a colon (:).
  • Property: The style attribute you want to change (e.g., color, font-size, margin).
  • Value: The value for the property (e.g., blue, 16px, 20px).
  • Each declaration ends with a semicolon (;).

3. CSS Selectors

Selectors are used to "find" (or select) the HTML elements you want to style.

Element Selector

Selects HTML elements based on their tag name.

p {
    color: #333;
    font-size: 16px;
}

ID Selector

Selects an element with a specific id attribute. IDs must be unique within an HTML document. Uses a # prefix.

#my-heading {
    text-align: center;
    color: #DF4881;
}

Class Selector

Selects elements with a specific class attribute. Classes can be used on multiple elements. Uses a . prefix.

.highlight {
    background-color: #F7C8DA;
    font-weight: bold;
}

Universal Selector

Selects all HTML elements on the page. Uses an asterisk (*).

* {
    margin: 0;
    padding: 0;
}

Grouping Selector

Selects multiple elements with the same style definitions. Separate selectors with a comma.

h1, h2, p {
    font-family: sans-serif;
}

4. Common CSS Properties

There are hundreds of CSS properties, but here are some of the most commonly used ones:

Color and Background

p {
    color: #333; /* Text color */
    background-color: #F8F5FF; /* Background color of the element */
}
body {
    background-image: url('https://placehold.co/1200x800/ADADAD/FFFFFF?text=Background'); /* Placeholder background image */
    background-repeat: no-repeat;
    background-size: cover;
}

Text Properties

h1 {
    font-family: 'Arial', sans-serif; /* Font typeface */
    font-size: 32px; /* Size of the font */
    font-weight: bold; /* Boldness of the text */
    text-align: center; /* Alignment of text */
    line-height: 1.5; /* Spacing between lines of text */
    text-decoration: underline; /* Underline, overline, line-through */
}

Box Model Properties (Padding, Border, Margin)

div {
    width: 200px; /* Width of the content area */
    height: 150px; /* Height of the content area */
    padding: 20px; /* Space between content and border */
    border: 2px solid #C430D7; /* Border around the content and padding */
    margin: 10px auto; /* Space outside the border */
}

5. The CSS Box Model

In CSS, every element is treated as a rectangular box. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

  1. Content: The actual content of the element, such as text or images.
  2. Padding: Clears an area around the content. It is transparent.
  3. Border: A border that goes around the padding and content.
  4. Margin: Clears an area outside the border. It is transparent.
.box-example {
    width: 200px;
    height: 100px;
    background-color: #F7C8DA; /* Content area background */
    padding: 20px; /* Space inside the border */
    border: 5px solid #DF4881; /* The border itself */
    margin: 30px; /* Space outside the border */
}

Understanding the box model is crucial for controlling the layout and spacing of elements on your web page.

6. Flexbox for Layout

Flexbox (Flexible Box Layout Module) is a one-dimensional layout method for arranging items in a container, allowing them to be aligned and distributed easily. It's excellent for creating responsive components.

Basic Flex Container

.flex-container {
    display: flex; /* Makes the container a flex container */
    justify-content: space-around; /* Distributes items with space around them */
    align-items: center; /* Aligns items vertically in the center */
    height: 150px;
    background-color: #E2B2EC;
    border: 1px dashed #C430D7;
    padding: 10px;
}
.flex-item {
    background-color: #DF4881;
    color: white;
    padding: 10px 20px;
    margin: 5px;
    border-radius: 5px;
}

And the corresponding HTML: 

<div class="flex-container">
    <div class="flex-item">Item 1</div>
    <div class="flex-item">Item 2</div>
    <div class="flex-item">Item 3</div>
</div>

Flexbox greatly simplifies complex layouts, especially for aligning and distributing items within a single row or column.
