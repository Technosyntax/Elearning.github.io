1. Introduction to Java

Java is a popular, high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It's known for its "Write Once, Run Anywhere" (WORA) capability, meaning compiled Java code can run on all platforms that support Java without the need for recompilation.

It's widely used for developing mobile applications (Android), web applications, desktop applications, games, and much more!

2. Java Development Environment

To write and run Java programs, you'll need the Java Development Kit (JDK). The JDK includes:

  • JRE (Java Runtime Environment): Contains the JVM (Java Virtual Machine) and class libraries needed to run Java applications.
  • JVM (Java Virtual Machine): The abstract machine that executes Java bytecode.
  • Development Tools: Compilers (javac), debuggers, and other tools necessary for developing Java applications.

You can download the JDK from Oracle's official website. After installation, make sure to set up your environment variables (PATH) correctly.

3. Basic Syntax and "Hello World!"

Let's start with the classic "Hello World!" program in Java. 

public class HelloWorld { // Class declaration
    public static void main(String[] args) { // Main method - entry point of the program
        System.out.println("Hello, World!"); // Prints the message to the console
    }
}
  • public class HelloWorld: Every Java application must have at least one class, and the filename must match the class name (HelloWorld.java).
  • public static void main(String[] args): This is the main method, the entry point for all Java applications.
  • System.out.println(): Used to print output to the console.

To compile: javac HelloWorld.java
To run: java HelloWorld

4. Variables and Data Types

Variables are containers for storing data values. In Java, you must declare the variable's type before using it.

Primitive Data Types:

  • int: Integer numbers (e.g., 10, -5)
  • double: Floating-point numbers (e.g., 3.14, 2.5)
  • boolean: True or false values (true, false)
  • char: Single characters (e.g., 'A', 'b')
  • String: (Not a primitive, but commonly used) Sequence of characters (e.g., "Hello", "Java")
int age = 30; // Declaring and initializing an integer variable
double price = 19.99; // Declaring and initializing a double variable
boolean isActive = true; // Declaring and initializing a boolean variable
char grade = 'A'; // Declaring and initializing a character variable
String name = "Alice"; // Declaring and initializing a String variable

System.out.println("Name: " + name + ", Age: " + age);

5. Operators

Operators are used to perform operations on variables and values.

Arithmetic Operators:

  • + (Addition), - (Subtraction), * (Multiplication), / (Division), % (Modulus)
int x = 10;
int y = 3;
System.out.println("Sum: " + (x + y)); // Output: Sum: 13
System.out.println("Remainder: " + (x % y)); // Output: Remainder: 1

Comparison Operators:

  • == (Equal to), != (Not equal to), > (Greater than), < (Less than), etc.
int a = 5;
int b = 8;
System.out.println(a == b); // Output: false
System.out.println(a < b); // Output: true

6. Control Flow (Conditional Statements & Loops)

Control flow statements enable Java to execute different blocks of code based on conditions or to repeat code.

If-Else Statement

int score = 75;
if (score >= 60) {
    System.out.println("Passed!");
} else {
    System.out.println("Failed.");
}

For Loop

for (int i = 0; i < 5; i++) {
    System.out.println("Loop iteration: " + i);
}
/* Output:
Loop iteration: 0
Loop iteration: 1
Loop iteration: 2
Loop iteration: 3
Loop iteration: 4
*/

While Loop

int count = 0;
while (count < 3) {
    System.out.println("Count is: " + count);
    count++;
}
/* Output:
Count is: 0
Count is: 1
Count is: 2
*/

7. Methods (Functions)

Methods are blocks of code that perform a specific task. They make code more organized, readable, and reusable. 

public class MyCalculations {
    // Method to add two integers
    public static int add(int num1, int num2) {
        return num1 + num2;
    }

    // Method with no return value (void)
    public static void greet(String name) {
        System.out.println("Hello, " + name + "!");
    }

    public static void main(String[] args) {
        int sum = add(5, 3); // Calling the add method
        System.out.println("Sum: " + sum); // Output: Sum: 8

        greet("Charlie"); // Calling the greet method
    }
}

8. Introduction to OOP (Classes & Objects)

Java is an object-oriented language. OOP is a programming paradigm based on the concept of "objects", which can contain data and code: data in the form of fields (attributes), and code in the form of procedures (methods).

  • Class: A blueprint or template for creating objects. It defines the properties (fields) and behaviors (methods) that objects of that class will have.
  • Object: An instance of a class. You can create multiple objects from a single class.
// Define a simple Car class
class Car {
    String color;
    String brand;

    // Method to display car details
    void displayDetails() {
        System.out.println("Brand: " + brand + ", Color: " + color);
    }
}

public class Main {
    public static void main(String[] args) {
        // Create an object of the Car class
        Car myCar = new Car();

        // Access attributes and assign values
        myCar.brand = "Toyota";
        myCar.color = "Blue";

        // Call a method on the object
        myCar.displayDetails(); // Output: Brand: Toyota, Color: Blue

        Car anotherCar = new Car();
        anotherCar.brand = "Honda";
        anotherCar.color = "Red";
        anotherCar.displayDetails(); // Output: Brand: Honda, Color: Red
    }
}
```
