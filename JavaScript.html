1. Introduction to JavaScript

JavaScript is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions. While it is most well-known as the scripting language for web pages, many non-browser environments also use it, such as Node.js, Apache CouchDB, and Adobe Acrobat.

It's primarily used to make web pages interactive, bringing static HTML and CSS to life. Think animations, interactive forms, dynamic content updates, and much more!

2. Including JavaScript in HTML

You can add JavaScript to your HTML documents in a few ways:

Inline JavaScript

Adding JavaScript directly within HTML attributes (generally discouraged for complex scripts).

<button onclick="alert('Hello from JavaScript!')">Click Me</button>

Internal JavaScript

Placing JavaScript code within <script> tags inside your HTML file (often in the <head> or just before the closing </body> tag).

<!-- Inside <head> or <body> -->
<script>
    console.log("Hello from internal JavaScript!");
</script>

External JavaScript (Recommended)

Linking to a separate .js file. This keeps your HTML clean and JavaScript reusable.

<!-- In your HTML head or before </body> -->
<script src="script.js"></script>

For external scripts, create a file named `script.js` (or any name you prefer) in the same directory as your HTML file, and put your JavaScript code there.

4. Variables and Data Types

Variables are containers for storing data values. JavaScript is dynamically typed, meaning you don't declare the type explicitly.

Declaring Variables:

  • var: Oldest way, function-scoped, can be re-declared and re-assigned. (Generally avoided in modern JavaScript)
  • let: Block-scoped, can be re-assigned but not re-declared in the same scope.
  • const: Block-scoped, cannot be re-assigned or re-declared (constant).

Common Data Types:

  • String: Text (e.g., "Hello", 'World')
  • Number: Numeric values (e.g., 10, 3.14, -5)
  • Boolean: Logical values (true, false)
  • Null: Intentional absence of any object value.
  • Undefined: A variable that has been declared but not assigned a value.
  • Object: Complex data structures (e.g., { key: 'value' }, arrays, functions).
let userName = "Alice"; // String
const age = 25; // Number
let isStudent = true; // Boolean
let score = null; // Null
let city; // Undefined

console.log(userName, age, isStudent, score, city);

5. Operators

Operators are symbols that perform operations on operands (values and variables).

Arithmetic Operators:

  • + (Addition), - (Subtraction), * (Multiplication), / (Division), % (Modulus)
let num1 = 10;
let num2 = 3;
console.log(num1 + num2); // 13
console.log(num1 % num2); // 1

Assignment Operators:

  • = (Assign), += (Add and assign), -= (Subtract and assign), etc.
let count = 5;
count += 2; // Same as: count = count + 2;
console.log(count); // 7

Comparison Operators:

  • == (Equal value), === (Equal value and type), != (Not equal value), !== (Not equal value or type), >, <, etc.
let x = 10;
let y = "10";
console.log(x == y);  // true (compares value)
console.log(x === y); // false (compares value AND type)

6. Control Flow (Conditionals & Loops)

Control flow statements dictate the order in which your code is executed.

If-Else If-Else Statement

let temperature = 28;
if (temperature > 30) {
    console.log("It's hot!");
} else if (temperature > 20) {
    console.log("It's warm.");
} else {
    console.log("It's cool.");
}

For Loop

for (let i = 0; i < 3; i++) {
    console.log("Iteration: " + i);
}
/* Output:
Iteration: 0
Iteration: 1
Iteration: 2
*/

While Loop

let i = 0;
while (i < 2) {
    console.log("While count: " + i);
    i++;
}
/* Output:
While count: 0
While count: 1
*/

7. Functions

Functions are blocks of code designed to perform a particular task. They allow you to reuse code.

Function Declaration

function greet(name) {
    return "Hello, " + name + "!";
}

console.log(greet("Bob")); // Output: Hello, Bob!

Arrow Functions (ES6+)

const multiply = (a, b) => a * b;

console.log(multiply(4, 5)); // Output: 20

8. DOM Manipulation

The **D**ocument **O**bject **M**odel (DOM) is a programming interface for web documents. It represents the page structure as a tree of objects, allowing JavaScript to access and manipulate HTML and CSS.

Selecting Elements:

// Select by ID
const myHeading = document.getElementById('main-heading');
console.log(myHeading.textContent); // Access text content

// Select by Class
const paragraphs = document.getElementsByClassName('my-paragraph');
console.log(paragraphs[0].textContent);

// Select by Tag Name
const allDivs = document.getElementsByTagName('div');

// More modern & versatile selectors (CSS-like)
const firstDiv = document.querySelector('.container div'); // Selects the first div inside .container
const allButtons = document.querySelectorAll('button'); // Selects all buttons

Modifying Elements:

// Change text content
myHeading.textContent = "New Heading Text";

// Change HTML content
const someDiv = document.getElementById('dynamic-content');
someDiv.innerHTML = '<p>This content was added by <strong>JavaScript</strong>.</p>';

// Change styles
myHeading.style.color = 'blue';
myHeading.style.backgroundColor = '#e0ffe0';

// Add/Remove/Toggle classes
myHeading.classList.add('font-bold');
myHeading.classList.remove('text-xl');
myHeading.classList.toggle('hidden'); // Toggles visibility

Event Handling:

Making elements respond to user actions (clicks, key presses, etc.).

const myButton = document.getElementById('my-button');

myButton.addEventListener('click', function() {
    alert('Button was clicked!');
});

// HTML for the button:
// <button id="my-button" class="bg-[#DF4881] text-white p-3 rounded-md">Click Me</button>
